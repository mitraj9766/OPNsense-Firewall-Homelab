# OPNsense-Firewall-Homelab

## Overview
This repository contains my hands-on OPNsense Firewall Homelab performed in VirtualBox. The project covers OPNsense installation, initial configuration, interface setup, firewall rule creation, alias configuration, ICMP filtering, connectivity testing using Kali Linux, and firewall log verification.

## Lab Environment
- OPNsense
- VirtualBox
- Kali Linux

## Topics Covered
- OPNsense Installation
- Initial Configuration
- Interface Configuration
- Web GUI Configuration
- Firewall Rule Configuration
- Firewall Aliases
- ICMP Blocking
- Connectivity Testing
- Firewall Log Analysis
